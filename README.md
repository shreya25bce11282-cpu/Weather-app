A simple Python desktop application that shows you the current weather information for any city, using the OpenWeatherMap API and a Tkinter GUI.

## Features

- Enter any city and instantly get:
  - Temperature (in Celsius)
  - Weather condition (clear, rain, etc.)
  - Country code
- User-friendly interface with modern design.
- Scrollable output area for searching multiple cities.

## How to Run

1. **Requirements:**
   - Python 3.x (tested on 3.13)
   - The `requests` package
   - The `tkinter` GUI toolkit (usually included with Python)
   - The `scrolledtext` module (part of tkinter)


2. **Setup the API Key:**
- Sign up at [OpenWeatherMap](https://openweathermap.org/) and get your free API key.
- Paste your API key in the `api_key` variable in the Python script.

3. **Run the app**
